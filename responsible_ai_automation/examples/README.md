# 예제 모음

이 디렉토리에는 Responsible AI Automation 시스템의 다양한 사용 예제가 포함되어 있습니다.

## 예제 목록

### 1. basic_usage.py
기본적인 사용 방법을 보여주는 예제입니다.

**실행 방법:**
```bash
python examples/basic_usage.py
```

**주요 내용:**
- 시스템 초기화
- 모델 등록 및 평가
- Responsible AI 지표 확인

### 2. custom_evaluator.py (예정)
커스텀 평가자를 만드는 방법을 보여주는 예제입니다.

### 3. monitoring_dashboard.py (예정)
모니터링 대시보드를 사용하는 방법을 보여주는 예제입니다.

### 4. rl_training.py (예정)
강화 학습 에이전트를 훈련시키는 방법을 보여주는 예제입니다.

## 예제 실행 전 확인사항

1. 필요한 패키지가 설치되어 있는지 확인하세요:
```bash
pip install -r requirements.txt
```

2. `config.yaml` 파일이 프로젝트 루트에 있는지 확인하세요.

3. 예제 실행:
```bash
cd examples
python basic_usage.py
```
