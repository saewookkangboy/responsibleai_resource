# 프로젝트 시작 전 체크리스트

이 체크리스트는 AI 프로젝트를 시작하기 전에 수행해야 할 AI 윤리 및 Responsible AI 관련 검토 사항을 포함합니다.

## 프로젝트 기획

### 목표 및 범위 정의
- [ ] 프로젝트 목표에 AI 윤리 원칙 명시
- [ ] Responsible AI를 프로젝트 성공 기준에 포함
- [ ] 프로젝트 범위에 공정성, 투명성, 프라이버시 요구사항 포함
- [ ] 프로젝트 제한사항 및 가정 명시

### 이해관계자 확인
- [ ] 주요 이해관계자 식별
- [ ] AI 윤리 위원회 또는 전문가와의 연계 체계 구축
- [ ] 이해관계자 브리핑 일정 수립
- [ ] 의사결정 권한 및 승인 프로세스 정의

### 위험 평가
- [ ] 잠재적 윤리적 위험 식별
- [ ] 위험 영향도 및 발생 가능성 평가
- [ ] 위험 완화 전략 수립
- [ ] 위험 레지스터 작성

## 데이터 계획

### 데이터 수집 계획
- [ ] 데이터 수집 목적 및 범위 명확화
- [ ] 데이터 출처의 합법성 및 윤리성 확인
- [ ] 데이터 수집 시 사용자 동의 필요성 검토
- [ ] 개인정보 보호법 준수 계획 (GDPR, 개인정보보호법 등)

### 데이터 품질 계획
- [ ] 데이터 대표성 및 다양성 검증 계획
- [ ] 데이터 편향 검증 계획
- [ ] 데이터 품질 메트릭 정의
- [ ] 데이터 보관 및 삭제 정책 수립

## 팀 구성

### 역할 및 책임
- [ ] 각 팀원의 AI 윤리 관련 책임 명시
- [ ] AI 윤리 전문가 또는 컨설턴트 포함 여부 확인
- [ ] 윤리 검토 프로세스 정의
- [ ] 윤리 관련 이슈 에스컬레이션 경로 명확화

### 교육 및 인식
- [ ] 팀원 AI 윤리 교육 계획 수립
- [ ] Responsible AI 원칙 및 가이드라인 공유
- [ ] 역할별 가이드라인 배포
- [ ] 정기적인 윤리 검토 미팅 일정 수립

## 일정 및 리소스

### 일정 계획
- [ ] 프로젝트 일정에 윤리 검증 단계 포함
- [ ] 각 단계별 윤리 체크포인트 설정
- [ ] 공정성 검증 및 완화를 위한 충분한 시간 배정
- [ ] 문서화 및 리뷰 시간 확보

### 리소스 계획
- [ ] 윤리 검증에 필요한 도구 및 라이브러리 확인
- [ ] 공정성 검증 도구 도입 계획
- [ ] 모니터링 및 로깅 시스템 계획
- [ ] 예산 및 리소스 할당

## 법규 및 규정 준수

### 법규 검토
- [ ] 관련 법규 및 규정 확인 (GDPR, 개인정보보호법 등)
- [ ] 산업별 규제 요구사항 확인
- [ ] 법무팀과의 협의 일정 수립
- [ ] 규정 준수 체크리스트 작성

### 윤리 가이드라인
- [ ] 회사 내부 AI 윤리 가이드라인 확인
- [ ] 산업 표준 및 모범 사례 검토
- [ ] 외부 윤리 가이드라인 참고 (IEEE, Partnership on AI 등)

## 승인 및 서명

### 초기 승인
- [ ] 프로젝트 착수 승인
- [ ] AI 윤리 위원회 초기 검토 완료
- [ ] 이해관계자 동의 확인
- [ ] 프로젝트 킥오프 미팅 완료

## 체크리스트 완료 확인

- [ ] 모든 항목 검토 완료
- [ ] 누락된 항목 보완
- [ ] 프로젝트 시작 승인 받음
- [ ] 체크리스트 문서화 및 보관
