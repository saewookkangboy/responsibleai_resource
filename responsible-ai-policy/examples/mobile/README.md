# 모바일 앱 예제

이 디렉토리에는 iOS와 Android 모바일 앱에 Responsible AI를 적용하는 예제가 포함되어 있습니다.

## 구조

- `ios/`: iOS (Swift) 예제
- `android/`: Android (Kotlin) 예제
- `react-native/`: React Native 크로스 플랫폼 예제

## 주요 기능

1. **안전한 API 통신**
   - HTTPS 강제
   - Certificate Pinning
   - API 키 안전한 저장

2. **프라이버시 보호**
   - Keychain/Keystore 사용
   - 로컬 데이터 암호화
   - 사용자 동의 관리

3. **AI 기능 투명성**
   - AI 사용 명시
   - 의사결정 설명
   - 사용자 제어 옵션

4. **데이터 관리**
   - 데이터 보기
   - 데이터 내보내기
   - 데이터 삭제
