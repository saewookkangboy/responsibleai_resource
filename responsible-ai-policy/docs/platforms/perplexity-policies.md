# Perplexity AI 정책 및 가이드라인

## 개요

Perplexity AI는 AI 기반 검색 엔진을 제공하는 기업으로, 사용자의 질문에 대해 인터넷을 검색하여 신뢰할 수 있는 출처를 기반으로 한 답변을 제공합니다.

## 핵심 원칙

### 1. 정보의 신뢰성
- **출처 명시**: 모든 답변에 신뢰할 수 있는 출처 제공
- **사실 검증**: 정확한 정보 제공을 위한 검증 프로세스
- **투명성**: 정보 수집 및 처리 과정의 투명성

### 2. 저작권 및 지적재산권 보호
- **콘텐츠 사용**: 저작권을 존중하는 콘텐츠 사용
- **출처 표기**: 적절한 출처 표기 및 인용
- **무단 사용 방지**: 저작권 침해 방지를 위한 정책 수립

### 3. 사용자 프라이버시
- **데이터 최소화**: 필요한 최소한의 데이터만 수집
- **익명화**: 사용자 식별 정보 보호
- **데이터 보안**: 안전한 데이터 처리 및 저장

## 주요 이슈 및 대응

### 1. 저작권 침해 문제
Perplexity는 여러 언론사로부터 저작권 침해 및 무단 콘텐츠 사용과 관련하여 법적 문제에 직면한 바 있습니다.

**대응 방안:**
- 저작권 보호를 위한 정책 강화
- 콘텐츠 사용 시 적절한 라이선스 확인
- 출처 명시 및 인용 규칙 준수

### 2. AI 응답 조작 가능성
AI 검색 엔진의 조작 가능성에 대한 우려가 제기되었습니다. '답변 엔진 최적화(AEO)'와 같은 문제가 발견되었습니다.

**대응 방안:**
- AI 응답의 신뢰성과 투명성 강화
- 조작 방지를 위한 검증 메커니즘
- 사용자에게 AI 사용 여부 명시

### 3. 정보의 정확성
AI 기반 검색 결과의 정확성을 보장하기 위한 노력이 필요합니다.

**대응 방안:**
- 다중 출처 검증
- 사실 확인 프로세스
- 오류 수정 메커니즘

## 서비스 개발에의 적용

### 웹 서비스
- 출처 명시 기능 구현
- 저작권 확인 프로세스
- 사용자에게 AI 사용 명시

### 모바일 앱
- 오프라인 검색 결과 캐싱
- 프라이버시 설정 옵션
- 출처 추적 기능

### API 서비스
- 출처 정보 포함 응답
- Rate limiting으로 남용 방지
- 저작권 검증 API

### 2. 책임성

- 사용자 피드백 채널
- 정기적인 정책 검토

### 3. 프라이버시

- 사용자 데이터 보호
- 명시적 동의 획득
